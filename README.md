This project analyzes eBay listing data to understand how different features such as product category, condition, bidding information and seller metrics influence the final selling price. I used Python in a Jupyter notebook to clean the raw dataset, explore key distributions and relationships, and build simple regression models to estimate item prices.
The notebook walks through the full workflow: loading the CSV data, handling missing values, engineering useful features and evaluating model performance using metrics like RMSE. The final report summarizes the main pricing insights and highlights which variables have the strongest impact on eBay sale prices, giving practical guidance for sellers on how to optimize their listings.
